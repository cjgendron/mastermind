Welcome to Mastermind Solver!

Background:

There are 32768 possible patterns that can be created in a game of Ultimate Mastermind, which has 5 spaces, and 8 possible colors in each space. When a player guesses a pattern, that pattern, when compared to a prior guess, must yield the same result of red and white pegs as the solution pattern for it to be a valid guess. This solver uses that approach to rapidly solve via brute force for the proper pattern. The solver begins by generating all 32768 patterns and giving the user a guess. The first guess is somewhat arbitrary, and in this case is hard coded to Red, Orange, Yellow, Green, Blue. As the user enters the red/white peg counts, the system continuously filters the list of possible patterns by removing patterns that would not yield those same counts when compared against the prior guess.

Directions:

This solver will allow you to solve a game of mastermind

  1. Enter the displayed "guess" into your actual game of Mastermind.
  2. Enter the number of red and white pegs that the guess received
  3. Click "Guess"
  4. Repeat until your guess is the right answer
Notes:
  1. As the number of possible answers decreases, the chances that the displayed guess is the correct pattern increases
  2. If the number of possible answers is 1, then the displayed guess MUST be the correct pattern - if it isn't then you either entered your guess incorrectly into the actual game or there is an error in the number of red and white pegs from a prior guess
  3. If the number of possible answers is 0, then there MUST be an error in a prior guess
